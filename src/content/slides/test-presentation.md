# Dark Matter Slides

A RevealJS presentation system with **three-mode theme support**

---

## Three Modes

- **Light Mode** - Clean, professional appearance
- **Dark Mode** - Easy on the eyes, modern look
- **Vibrant Mode** - Enhanced glows and effects

*Use the star/moon/sun toggle to switch!*

---

## Features

Built on RevealJS 4.5 with:

- Adaptive typography using `clamp()`
- Responsive controls for all screen sizes
- Keyboard navigation
- Speaker notes support

---

## Keyboard Shortcuts

- **Arrow keys** - Navigate slides
- **F** - Toggle fullscreen
- **S** - Open speaker notes
- **ESC** - Slide overview
- **B** - Pause presentation

---

## Code Support

Inline code: `const theme = 'matter'`

---

## Try the Modes!

Click the **mode toggle** button in the control bar to cycle through:

1. Light Mode (Sun icon)
2. Dark Mode (Moon icon)
3. Vibrant Mode (Star icon)
